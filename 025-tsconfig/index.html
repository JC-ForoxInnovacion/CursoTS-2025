Backup tsconfig.json

Antes de realizar cualquier cambio, como medida de seguridad, deberías hacer una copia de seguridad del archivo tsconfig.json

YYYY-MM-DD_HH.MM-tsconfig.json.txt

Para restaurarla, borra el tsconfig.json y renombra el archivo de backup que has creado.

Documentación oficial tsconfig

1. Depuración con TS

Las refencias a líneas de código en consola se muestran con relación al archivo transpilado a .js, pero no coinciden con el número de línea del archivo .ts

Para referenciar correctamente los números de línea al archivo.ts activar esta la configuración: 

              "sourceMap": true,

2. Eliminar comentarios en archivo .js

Para eliminar los comentarios del archivo.ts al transpilar a .js, activar la siguiente configuración: 

              "removeComments": true,

3. Incluir / Excluir carpetas y/o archivos

Para incluir y/o excluir carpetas y/o archivos debemos añadir las propiedades "include": [] / "exclude": [] despues del cierre de objeto principal. 

              {
                "compilerOptions": {
                  /* Visit https://aka.ms/tsconfig to read more about this file */
              
                  /* Projects */
                  // "incremental": true, 

                  ...

                  /* Completeness */
                  // "skipDefaultLibCheck": true,    /* Skip type checking .d.ts files ... */
                  "skipLibCheck": true               /* Skip type checking all .d.ts files ... */
                },
                "include": [],
                "exclude": [
                  "my_node_modules"
                ]
              }
